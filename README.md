# SUPPORT MAG

### 📋 Objectifs 
__________
Contiens les outils et scripts utiles au support magasin.

### ✅ Bonnes pratiques
_______

Tous les scrips doivent contenir une entête :

- `Fichiers SQL` : 
```SQL
/*
    .SYNOPSIS
    .DESCRIPTION
    .REQUIREMENTS
    .NOTES
        Nom du fichier :
        Auteur : 
        Date   : 
        Version: 
*/
```
- `Fichiers PS` : 

```powershell
<#
    .SYNOPSIS
    .DESCRIPTION
    .PARAMETER <Parameter>
    .OUTPUTS
    .INPUTS 
    .EXAMPLE
    .NOTES 
        Nom du fichier :
        Auteur : 
        Date   : 
        Version: 
#>
