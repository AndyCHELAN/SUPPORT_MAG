# SUPPORT MAG

### 📋 Objectifs 
__________
Contiens les outils et scripts utiles au support magasin.

### ✅ Bonnes pratiques
_______

Tous les scrips doivent contenir une entête :

- `Fichiers SQL` : 
```SQL
/*
	.DESCRIPTION
		Règles de contrôle :
		Environnement :
	.PREREQUISITES
	.NOTE
		Fichier :
		Auteur  :
        Version :
	.HISTORIQUE
		+---------+------------+--------------+------------------------------------------+
		| Version | Date       | Auteur       | Modifications                            |
		+---------+------------+--------------+------------------------------------------+
		| 1.0     |            |              | Création du script                       |
		+---------+------------+--------------+------------------------------------------+
*/
```
- `Fichiers PS` : 

```powershell
<#
    .SYNOPSIS
    .DESCRIPTION
    .PARAMETER <Parameter>
    .OUTPUTS
    .INPUTS 
    .EXAMPLE
    .NOTES 
        Prérequis :
        Fichier :
        Version : 
        Historique :
        +---------+------------+--------------+------------------------------------------+
		| Version | Date       | Auteur       | Modifications                            |
		+---------+------------+--------------+------------------------------------------+
		| 1.0     |            |              | Création du script                       |
		+---------+------------+--------------+------------------------------------------+
#>
```
